# Enhancing-Large-Language-Models-LLAMA-QWEN-Efficiency-Through-Layer-Pruning
A framework to enhance LLM efficiency by combining layer pruning and speculative decoding. This tool analyzes models like Llama 3 &amp; Qwen to find redundant layers, enabling the pruned model to act as a high-fidelity ‘drafter’ for accelerated, lossless inference.
